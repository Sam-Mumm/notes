# Vault

## Erstellen
```
$ ansible-vault create <file>.yml
```

## Syntax im Vault


## Verschlüsseln / Entschlüsseln
```
# Entschlüsseln
$ ansible-vault encrypt <file>.yml

# Verschlüsseln
$ ansible-vault decrypt <file>.yml
```

## Bearbeiten
```
$ ansible-vault edit <file>.yml
```
