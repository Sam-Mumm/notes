# Optionen
| -- | -- |
|``set bg=dark``|Umschalten bei dunklen Hintergrund|
|``set number``|Einschalten der Zeilenummern|
|``set nonumber``|Ausschalten der Zeilennummern|
|``set list``|Anzeigen von nicht-druckbaren Zeichen|
